# ESRPCB

If you want to retrain the model, please download the dataset from [here](https://www.kaggle.com/datasets/norbertelter/pcb-defect-dataset)

The structure of the dataset will be like this:
```
dataset/
    train/
        images/
        labels/
    val/
        images/
        labels/
    test/
        images/
        labels/

```
